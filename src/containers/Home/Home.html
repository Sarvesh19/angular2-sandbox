This is my sandbox project on which I experiment on Angular2 and RxJS, using TypeScript and Webpack.

I tried to avoid doing the TodoList we tend to always do when trying up new frameworks, you'll find an example with "Color Interval" where I mix up Angular2, RxJS and @ngrx/store.

Try Color Interval

I've used my own boilerplate topheman/webpack-babel-starter that I ported from Babel/ESLint to TypeScript/TSLint to bootstrap this project.

If you like this project, take a look at topheman/rxjs-experiments, an RxJS implementation of multi-touch / accelerometer.

This is the production packaged version (html/css/js uglified). You can test the site in debug mode here, where you'll have access to:

  • sourcemaps for scss/ts

I'm a developer, I want to see what's under the hood!

This is the development packaged version (the production version is here). In the current mode, you have access to:
